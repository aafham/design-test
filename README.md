# design-test

iOS 26-inspired "Liquid Glass" marketing + ordering UI concept for a boutique cake shop.

## Current Stack

This project is plain static web:

- `index.html`
- `styles.css`
- `app.js`

No framework or build tool is required.

## Highlights

- Light/dark theme with localStorage persistence
- iOS-like compact theme switch
- Sticky header with active nav section state
- Glass UI with controlled blur for performance
- Product/testimonial card clamping for readability
- WhatsApp floating CTA (desktop + mobile)
- Accessible focus ring + reduced-motion support

## Run Locally

Open `index.html` directly in your browser.
